Pentingnya pivot table 
-Mengolah Data dengan Cepat
Pivot Table memungkinkan mengelompokkan, menghitung total, rata-rata, atau membuat ringkasan data tanpa harus menulis rumus panjang.
-Sangat membantu saat menangani ribuan baris data.
Membuat Laporan Dinamis
Memudahkan pembuatan laporan yang dapat diubah-ubah hanya dengan drag-and-drop, tanpa mengubah data asli.
-Analisis Multidimensi
Bisa menganalisis data dari berbagai sudut pandang (misalnya penjualan berdasarkan bulan, produk, wilayah, dan lainnya) dalam hitungan detik.
-Integrasi dengan Visualisasi
Pivot Table dapat langsung diubah menjadi Pivot Chart untuk menampilkan data dalam bentuk grafik interaktif.

contoh penggunaan dalam data analyst
Menghitung total penjualan per bulan dan per produk.
-Menemukan jumlah pelanggan unik per kota.
-Menganalisis tren pertumbuhan dengan filter dinamis.
-Membuat dashboard sederhana tanpa perlu coding.

Di dalam pivot tabel ada alat yang sangat berguna untuk menganalisis data dan merangkum data.
-Filters (Filter):
Filters digunakan untuk menyaring data yang ditampilkan dalam Pivot Table. Dengan menggunakan filter, Anda dapat memilih untuk menampilkan hanya data tertentu berdasarkan kriteria yang Anda tentukan. Misalnya, jika Anda memiliki data penjualan dari berbagai wilayah, Anda dapat menggunakan filter untuk hanya menampilkan data dari satu wilayah tertentu.
-Columns (Kolom):
Kolom dalam Pivot Table digunakan untuk menampilkan kategori data secara horizontal. Misalnya, jika Anda menganalisis penjualan berdasarkan produk, Anda dapat menempatkan nama produk di bagian kolom untuk melihat penjualan masing-masing produk dalam format tabel.
-Rows (Baris):
Rows digunakan untuk menampilkan kategori data secara vertikal. Dalam contoh yang sama, Anda dapat menempatkan nama bulan di bagian baris untuk melihat penjualan per bulan. Dengan cara ini, Anda dapat dengan mudah membandingkan data di antara kategori yang berbeda.
-Values (Nilai):
Values adalah area di mana data numerik yang ingin Anda analisis ditampilkan. Ini bisa berupa jumlah, rata-rata, atau perhitungan lainnya dari data yang ada. Misalnya, jika Anda ingin melihat total penjualan, Anda akan menempatkan kolom penjualan di area Values.

Di Pivot Table Excel, fungsi-fungsi seperti SUM, AVERAGE, MAX, dan MIN juga digunakan untuk menganalisis dan merangkum data.
-SUM: Dalam Pivot Table, Anda dapat menggunakan fungsi SUM untuk menjumlahkan nilai dari suatu kolom berdasarkan kategori tertentu. Misalnya, jika Anda memiliki data penjualan, Anda dapat menjumlahkan total penjualan untuk setiap produk.
-AVERAGE: Fungsi AVERAGE di Pivot Table digunakan untuk menghitung rata-rata nilai dari suatu kolom berdasarkan kategori. Misalnya, Anda dapat menghitung rata-rata penjualan per bulan atau per produk.
-MAX: Fungsi MAX dalam Pivot Table digunakan untuk menemukan nilai maksimum dari suatu kolom berdasarkan kategori. Misalnya, Anda dapat menemukan penjualan tertinggi untuk setiap produk.
-MIN: Fungsi MIN di Pivot Table digunakan untuk menemukan nilai minimum dari suatu kolom berdasarkan kategori. Misalnya, Anda dapat menemukan penjualan terendah untuk setiap produk.

Cara menggunakan fungsi di pivot table:
-Membuat Pivot Table: Pilih data Anda, lalu pergi ke tab "Insert" dan pilih "PivotTable". Pilih lokasi untuk Pivot Table baru.
-Menambahkan Data: Seret kolom yang ingin Anda analisis ke area "Values" di Pivot Table Field List.
-Mengatur Fungsi: Klik pada dropdown di area "Values" dan pilih "Value Field Settings". Di sini, Anda dapat memilih fungsi yang ingin digunakan (SUM, AVERAGE, MAX, MIN, dll.).
-Menyesuaikan Tampilan: Anda dapat menambahkan kategori ke area "Rows" atau "Columns" untuk mengelompokkan data sesuai kebutuhan.
